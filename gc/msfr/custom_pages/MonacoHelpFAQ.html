Q : Comment télécharger et installer Windows 10 ?

R : Vous serez invité à télécharger et à installer Windows 10 après l’achat de votre licence. Cependant, si vous devez télécharger notre produit de nouveau ou si le téléchargement a été interrompu, suivez ces étapes.

Pour télécharger et installer Windows 10 :

  1. Identifiez-vous avec votre compte Microsoft.
  2. Dans le menu, sélectionnez Contenu numérique.
  3. Localisez votre achat de Windows 10, puis cliquez sur Télécharger.
  4. Suivez les instructions pour terminer d’installer Windows 10.
  5. À l’invite, vous devrez saisir votre clé de produit qui est aussi située dans votre Contenu numérique dans votre compte.

Q : Où trouver ma clé de produit ?

R : Votre clé de produit sera fournie avec votre achat. Cependant, si vous devez la fournir à nouveau, elle est incluse dans votre courrier électronique de confirmation ou dans votre Contenu numérique.

Pour trouver votre clé de produit :

  1. Identifiez-vous avec votre compte Microsoft.
  2. Dans le menu, sélectionnez Contenu numérique.
  3. Localisez votre achat Windows 10 et vous pourrez afficher votre clé de produit

Q : Et si j’ai des questions supplémentaires sur le téléchargement et l’installation de Windows 10 ?

R : Consultez la page de FAQ sur le téléchargement du logiciel Windows 10 pour plus de détails.


Les fonctionnalités de Windows 10 ne sont pas toutes compatibles avec tous les appareils. Assurez-vous de vérifier auprès de votre fabricant avant l’achat. Des détails supplémentaires sur les conditions de mise à niveau sont indiquées ci-dessous et sur la page des spécifications de Windows 10.

Q : Comment vérifier le statut de ma commande ?

R : Pour vérifier le statut de votre commande :

1. Identifiez-vous avec votre compte Microsoft.
2. Dans le menu latéral, cliquez sur Historique de commande.
3. Sélectionnez votre commande, puis cliquez sur Suivre le colis.


Q : Que faire si je n’ai pas reçu de facture ou de courriel de confirmation de mon achat sur le Microsoft Store ?

R : Parfois, votre dossier de courrier indésirable peut masquer des courriels importants par erreur, veuillez donc vérifier dans celui-ci si votre facture ne s’y trouve pas, au cas où elle n’apparaitrait pas dans votre boîte de réception.

Si vous ne trouvez toujours pas votre facture ou votre courriel de confirmation :

1. Identifiez-vous avec votre compte Microsoft.
2. Dans le menu latéral, cliquez sur Historique de commande, puis recherchez les détails de l’achat.

Remarque : Si vous avez reçu un message indiquant que le traitement de votre commande pourrait prendre jusqu’à 24h, ou si votre commande est marquée comme « en attente » dans votre historique de commande, cela signifie qu’elle est en cours d’analyse par notre Équipe de vérification des commandes dans le cadre de notre processus d’assurance qualité. Une fois l’analyse de votre commande terminée, celle-ci sera traitée ou vous serez informé en cas de problème.

Si vous avez besoin d’une assistance supplémentaire, n’hésitez pas à nous contacter au 09 70 01 90 90


Q : Que faire en cas de problème pour télécharger le produit que j’ai acheté sur le Microsoft Store ?

R : Visitez simplement notre Page de téléchargements de logiciels pour obtenir des conseils utiles et des instructions pour le téléchargement de nos produits. Vous pouvez aussi nous contacter au 08 05 98 00 79 si vous avez besoin d’assistance supplémentaire.


Q : Où puis-je obtenir de l’aide pour installer ou utiliser mon produit Microsoft ?

R : Visitez simplement http://support.microsoft.com , sélectionnez le produit pour lequel vous demandez de l’assistance, puis sélectionnez le sujet ou la catégorie qui décrit le mieux votre question ou votre problème.


Q : Comment puis-je obtenir de l’aide pour un PC ou une tablette doté de Microsoft Signature et acheté sur le Microsoft Store ?

R : Contactez-nous simplement au 08 05 98 00 79, sélectionnez l’option d’Assistance Signature et nous nous ferons un plaisir de vous aider.


Q : Quelle est la procédure à suivre pour retourner un produit acheté sur le Microsoft Store ?

R : Si vous avez acheté votre produit en ligne à travers le Microsoft Store , visitez simplement la page de retours et échanges.


Q: Votre produit bénéficie-t-il d'une garantie ?

R : De nombreux matériels de Microsoft Store sont fournis avec des garanties du fabricant. Pour certains produits, il est possible d’acheter un plan d’extension de garantie ou un plan contre les dommages accidentels contre paiement de frais supplémentaires. Ces garanties commerciales sont en supplément de tous les droits des clients ou garanties que vous pourriez obtenir dans le cadre de la loi. Veuillez consulter les pages du produit pour obtenir des informations détaillées spécifiques à chaque produit. Pour plus d’informations, veuillez consulter À propos des garanties clients et des droits conférés par la loi


Q : Comment puis-je supprimer mes informations de paiement ?

R : Les informations de paiement sont enregistrées sur votre compte Microsoft. Pour mettre à jour ou supprimer vos détails de paiement, consultez le site https://commerce.microsoft.com/PaymentHub/ et rendez-vous sur la page Options de paiement.
